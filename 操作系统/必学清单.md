# 一、定义
操作系统（OS） 是连接电脑硬件与应用软件的中间层系统软件，是整台设备的总管家。
> 没有操作系统，CPU、内存、硬盘只是一堆零件，无法运行任何软件。

# 二、核心作用
1. **管理硬件**
统一调度 CPU、内存、硬盘、显卡、网卡、键鼠等所有硬件资源，分配算力与空间。
2. **管理软件**
管控所有 APP、程序的启动、运行、暂停、关闭，避免程序互相抢占资源崩溃。
3. **提供接口**
给程序员 / 应用软件提供统一调用接口，不用直接操控底层硬件，大幅降低开发难度。
4. **人机交互**
图形桌面、鼠标点击、命令行，让普通人能轻松操作设备。

# 三、四大核心功能
1. **进程管理**
进程 = 正在运行的程序（打开浏览器、微信都是进程）
系统负责调度进程，决定哪个程序先用 CPU，实现多软件同时运行。
2. **内存管理**
给运行中的程序分配内存空间，闲置时回收内存，防止内存溢出、卡顿。
3. **文件管理**
统一管理硬盘文件，建立文件夹、读写、删除、分类存储数据。
14 **设备管理**
依靠硬件驱动识别外设（打印机、耳机、U 盘），统一调配使用。

# 四、层级结构（从上到下）
1. 应用软件：微信、浏览器、游戏、办公软件
2. 操作系统：系统桌面、系统工具、系统接口
3. 硬件驱动：让系统识别显卡、声卡、网卡等硬件
4. 系统内核：操作系统最核心，管控全部硬件资源
5. 物理硬件：CPU、内存、硬盘、主板
# 六、通俗比喻
电脑 = 一座工厂
- CPU = 生产车间
- 内存 = 临时工作台
- 硬盘 = 仓库
- 操作系统 = 工厂总管
总管分配车间算力、安排工作台、调度仓库货物，统筹所有工序；
各类软件就是干活的工人，只需要听从总管安排即可。

# 七、关键小知识点
- 并发：宏观同时开多个软件，微观 CPU 快速切换执行
- 虚拟内存：内存不够时，挪用部分硬盘空间临时充当内存
- 内核：操作系统灵魂，决定系统流畅度、功耗、性能
- 权限管理：区分管理员 / 普通用户，保护系统安全
